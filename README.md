# LinuxVPNKillswitch
**vswitch**: "VPN kill Switch"  
A Linux VPN killswitch designed, primarily, for OpenVPN via NordVPN hosting.  

This script was written natively on Solus OS built with compatibility in-mind for Debian based systems.
